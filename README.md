# CS5960PHYS
Everything to do with my master's thesis at UiO during the 25/26 school year.

## Repository structure

```text
CS5960PHYS/
├── archive/                        # Old versions of files and experiments that are not in use
├── figures/                        # Plots used in text
├── notebooks/                      # Jupyter notebooks I used for exploration and testing
├── main.py                         # Runs simulations, generates all plots and diagrams used
├── src/                            # Final experiment code that generates results and plots
│   ├── panqec_simulations/         # Simulations of quantum memory and some logical operators
│   ├── smaller_codes/              # Proof-of-concept experiments with the 5- and 7-qubit codes
│   └── t-gate_implementation/      # Implementation of T-gate
├── thesis/                         # Thesis manuscript sources (LaTeX)
├── pyproject.toml                  # uv environment specification
└── README.md                       # Project overview, setup, and usage instructions
```
