# CS5960PHYS
Everything to do with my master's thesis at UiO during the 25/26 school year.

## Repository structure

```text
CS5960PHYS/
├── archive/                        # Old versions of files and experiments
├── data/                           # Simulation output (JSON); not tracked in git, regenerated by main.py
├── figures/                        # Plots used in text
├── notebooks/                      # Jupyter notebooks I used for exploration and testing
├── src/                            # Final experiment code that generates results and plots
│   ├── panqec_simulations/         # Simulations of quantum memory and logical operators
│   │   ├── memory_simulation.py    # Quantum memory threshold experiment
│   │   ├── plotting.py             # All plot functions
│   │   ├── BBcode_classes.py       # Bivariate Bicycle code implementations
│   │   ├── decoder_classes.py      # BP-LSD decoder
│   │   └── errormodel_classes.py   # Gaussian Pauli error model
│   ├── smaller_codes/              # Proof-of-concept experiments with the 5- and 7-qubit codes
│   └── t-gate_implementation/      # Implementation of T-gate
├── thesis/                         # Thesis manuscript sources (LaTeX)
├── README.md                       # Project overview, setup, and usage instructions
├── main.py                         # Runs simulations, generates all plots and diagrams
└── pyproject.toml                  # uv environment specification

```

## How to run code

From the repository root:

```bash
python main.py
```

This runs all experiments and writes plots to `figures/`. Simulation data is cached to `data/` (not tracked in git) and reused on subsequent runs unless deleted.

## Thesis word count

From the repository root:
